Try the migration

  1. Confirm edge://flags/#web-app-migration-api is Enabled (on by default).
  2. Install this old app (button above, or the address-bar install icon).
  3. Open and install the new app.
  4. Launch this old app from your OS. With "behavior": "suggest" you’ll see a “Review app update” prompt in the app window offering to move to the new app.
  5. Inspect state at edge://web-app-internals and edge://apps.
